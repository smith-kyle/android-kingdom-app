# Find Quests Faster
<p>Life as a knight can be busy. Between damsel saving, dragon slaying, and exploring distant lands it may seem impossible to stay up-to-date with the quests in your area. <strong>Now there's an app that can help!</strong></p>
<img src="http://kylesmiff.com/images/4screens.png" width="100%">
<h2>Features</h2>
<ul>
	<li>Sign up</li>
	<li>Browse kingdoms</li>
	<li>Search through kingdoms based on name</li>
	<li>View quests based on kingdom location</li>
</ul>
